# 🏨 Secure Housekeeping & Inventory System

A professional Python CLI application designed to streamline hotel operations. This project demonstrates Object-Oriented Programming (OOP), Data Persistence, and Secure Authentication.

## 🔑 Key Features
- **User Authentication**: Secure Login/Register system with password confirmation and password change functionality.
- **RBAC (Role-Based Access Control)**: Permission layers for 'Admin' and 'Staff' users to protect sensitive data.
- **Inventory Logic**: Automated stock tracking, search functionality, and low-stock alerts.
- **Room Management**: OO-based room tracking that logs cleaning status and supply consumption.
- **JSON Persistence**: All data is serialized and stored in local JSON databases.

## 💻 Tech Stack
- **Language:** Python 3.x
- **Storage:** JSON (Structured Data)
- **Architecture:** Procedural & Object-Oriented

## 🚀 Getting Started
1. Run `app.py`.
2. Register a new user as an 'Admin' to access full deletion privileges.
3. Manage rooms and inventory through the interactive menus.
